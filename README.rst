MModel
======

`MModel` is a lightweights and modular model building framework
for small scale non linear models.

Installation
============

To view the graph, Graphviz needs to be installed:
`Graphviz Installation <https://graphviz.org/download/>_`

For development run:
``pip install .[test]``
